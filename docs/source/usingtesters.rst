==================
How to Use Testers
==================
.test(x) files
++++++++++++++

In order to use the testers provided you need to have appropriate test configuration files in the configuration
directory. Each test type has it's own file format, although most formats will have a special line near the top that
identifies the type of tester it goes to. The test configuration files need to end with .test or .testx (used for xml)
